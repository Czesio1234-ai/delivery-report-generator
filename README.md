# Delivery Report Generator (Python)

A simple Python project that reads delivery data from a CSV file and generates a summary report.
The project demonstrates basic Python scripting, data processing, and automation concepts.

## Features
- Reads delivery data from `deliveries.csv`
- Calculates total deliveries, total kilometers, and total delivery time
- Generates per-driver summaries
- Saves the output to `report.txt`
- Prints the report to the console

## Technologies
- Python 3
- CSV file handling
- Basic data aggregation

## How to run the project
1. Make sure Python 3 is installed
2. Clone the repository or download it as ZIP
3. Run the script:

```bash
python app.py
